# Детальная реферальная система

## ✅ Реализовано

### 1. API Endpoint
- **URL:** `GET /api/users/:id/referrals`
- **Функционал:**
  - Получение рефералов по всем 3 уровням
  - Построение реферального дерева
  - Расчет депозитов каждого реферала
  - Расчет заработка от каждого реферала
  - Статистика по уровням

### 2. Реферальное дерево
- **Компонент:** `ReferralTree.tsx`
- **Функционал:**
  - Визуализация иерархии рефералов
  - Раскрывающиеся узлы для просмотра дочерних рефералов
  - Отображение информации о каждом реферале:
    - Имя и username
    - Уровень в системе
    - Общая сумма депозитов
    - Количество транзакций
    - Заработок от реферала (TON и PLT бонусы)
  - Детальный список депозитов каждого реферала
  - Переход на страницу реферала по клику

### 3. Статистика по уровням

#### Уровень 1 (Прямые рефералы)
- Количество рефералов
- Общая сумма депозитов
- Общий заработок от уровня

#### Уровень 2 (Рефералы рефералов)
- Количество рефералов
- Общая сумма депозитов
- Общий заработок от уровня

#### Уровень 3 (Рефералы рефералов рефералов)
- Количество рефералов
- Общая сумма депозитов
- Общий заработок от уровня

### 4. Два режима отображения

#### Реферальное дерево
- Визуальное представление иерархии
- Раскрывающиеся узлы
- Детальная информация при раскрытии
- Навигация к страницам рефералов

#### Список по уровням
- Табличное представление
- Группировка по уровням
- Детальная информация о каждом реферале:
  - Пользователь (имя, ID)
  - Реферальный код
  - Сумма депозитов
  - Количество транзакций
  - TON и PLT бонусы
  - Дата регистрации

### 5. Детальная информация о рефералах

Для каждого реферала отображается:
- **Основная информация:**
  - ID пользователя
  - Username
  - Полное имя
  - Реферальный код
  - Уровень в системе
  - Дата регистрации

- **Финансовая информация:**
  - TON баланс
  - PLT баланс
  - Общая сумма депозитов
  - Количество депозитов
  - Список всех депозитов с датами

- **Заработок от реферала:**
  - TON бонус
  - PLT бонус

### 6. Общая статистика

В верхней части страницы отображается:
- Всего рефералов (по всем уровням)
- Общий заработок (по всем уровням)
- Общая сумма депозитов всех рефералов
- Реферальный код пользователя

## 🎨 Дизайн

- **Цветовая индикация уровней:**
  - Уровень 1: Primary цвет (синий)
  - Уровень 2: Accent цвет
  - Уровень 3: Muted цвет

- **Интерактивность:**
  - Раскрытие/сворачивание узлов дерева
  - Переход на страницу реферала по клику
  - Детальная информация при раскрытии

## 🔍 Логика построения дерева

1. **Уровень 1:** Пользователи, у которых `JoinedByCode` равен реферальному коду пользователя
2. **Уровень 2:** Пользователи, у которых `JoinedByCode` начинается с кода реферала уровня 1
3. **Уровень 3:** Пользователи, у которых `JoinedByCode` начинается с цепочки кодов уровней 1 и 2

Пример:
- Пользователь A (код: 123)
- Реферал уровня 1: B (JoinedByCode: "123")
- Реферал уровня 2: C (JoinedByCode: "123.456", где 456 - код B)
- Реферал уровня 3: D (JoinedByCode: "123.456.789", где 789 - код C)

## 🚀 Использование

1. Откройте страницу пользователя (`/users/:id`)
2. Перейдите на вкладку "Рефералы"
3. Выберите режим отображения:
   - **Реферальное дерево** - для визуального просмотра иерархии
   - **По уровням** - для детального анализа по уровням
4. Раскройте узлы дерева для просмотра дочерних рефералов
5. Кликните на реферала для перехода на его страницу
