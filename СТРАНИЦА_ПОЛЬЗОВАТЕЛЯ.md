# Страница пользователя - Детальная информация

## ✅ Создана страница детального просмотра пользователя

### Маршрут
- **URL:** `/users/:id`
- **Компонент:** `UserDetailPage.tsx`

### Функциональность

#### 1. **Информация о пользователе**
- Основная информация (ID, username, полное имя)
- Реферальный код
- Кошелёк TON
- Язык интерфейса
- Дата регистрации
- Последний вход
- Количество входов
- Статус бота

#### 2. **Балансы**
- TON баланс
- PLT баланс
- Виртуальный TON баланс

#### 3. **Депозиты**
- Полный список всех депозитов пользователя
- Информация о каждой транзакции:
  - ID заказа
  - Сумма и валюта
  - Статус (выполнено, в процессе, отменено)
  - Кошелёк отправителя
  - Дата создания

#### 4. **Выводы**
- Полный список всех выводов пользователя
- Информация о каждой транзакции:
  - ID заказа
  - Сумма и валюта
  - Статус
  - Кошелёк получателя
  - Дата создания

#### 5. **Позиции**
- Список всех инвестиционных позиций пользователя
- Информация о каждой позиции:
  - Валюта
  - Размер депозита
  - Статус (активна, завершена)
  - Интервал итерации
  - Последняя активация
  - Дата создания

#### 6. **Реферальная система**
- Общая статистика:
  - Всего рефералов
  - Заработано по реферальной программе
  - Реферальный код
- Детальный список рефералов:
  - ID каждого реферала
  - TON бонус от реферала
  - PLT бонус от реферала

#### 7. **Настройки**
- Реферальный код (с возможностью копирования)
- Кошелёк TON
- Язык интерфейса
- Статус активности пользователя

### Навигация

#### Из списка пользователей:
- Клик на строку пользователя → переход на детальную страницу
- Клик на кнопку "More" (три точки) → переход на детальную страницу

#### На странице пользователя:
- Кнопка "Назад" → возврат к списку пользователей

### Статистика на странице

В верхней части страницы отображаются карточки со статистикой:
- TON Баланс
- PLT Баланс
- Количество рефералов
- Заработано по реферальной программе

### Табы (вкладки)

Страница организована с помощью табов для удобной навигации:
1. **Информация** - основная информация и балансы
2. **Депозиты** - список всех депозитов
3. **Выводы** - список всех выводов
4. **Позиции** - инвестиционные позиции
5. **Рефералы** - реферальная система
6. **Настройки** - настройки пользователя

### API Endpoints

Используются следующие API endpoints:
- `GET /api/users/:id` - получение информации о пользователе
- `GET /api/transactions` - получение всех транзакций (фильтруются на клиенте)
- `GET /api/positions` - получение всех позиций (фильтруются на клиенте)

### Дизайн

- Используется тот же дизайн, что и на остальных страницах
- Glass-card стиль для всех блоков
- Адаптивная верстка
- Иконки для каждого раздела
- Цветовая индикация статусов

## 🚀 Готово к использованию!

Страница полностью функциональна и готова к использованию. Все данные загружаются из MongoDB в реальном времени.
