### API Endpoints (все работают):

1. **Users:**
   - `GET /api/users` - Список пользователей
   - `GET /api/users/:id` - Пользователь по ID
   - `GET /api/users/stats/summary` - Статистика

2. **Transactions:**
   - `GET /api/transactions` - Список транзакций
   - `GET /api/transactions/:id` - Транзакция по ID
   - `GET /api/transactions/stats/summary` - Статистика

3. **Positions:**
   - `GET /api/positions` - Список позиций
   - `GET /api/positions/stats/summary` - Статистика

4. **Tasks:**
   - `GET /api/tasks` - Список задач
   - `GET /api/tasks/claimed` - Выполненные задачи

5. **Referrals:**
   - `GET /api/referrals/levels` - Уровни рефералов
   - `GET /api/referrals/stats` - Статистика

6. **Settings:**
   - `GET /api/settings` - Получить настройки
   - `PUT /api/settings` - Обновить настройки

7. **Stats:**
   - `GET /api/stats/dashboard` - Статистика дашборда
