## 📋 Как открыть дашборд

1. **Откройте браузер** и перейдите по адресу:
   - http://localhost:5173
   - Или проверьте консоль PowerShell, где запущен frontend - там будет указан точный URL

2. **Если порт отличается**, проверьте вывод в консоли frontend сервера

## 🔧 Управление серверами

### Остановка серверов:
- Закройте окна PowerShell, где запущены серверы
- Или нажмите `Ctrl+C` в каждой консоли

### Перезапуск:
- Backend: `cd user-hub-dashboard-f5c43c42/server && npm run dev`
- Frontend: `cd user-hub-dashboard-f5c43c42 && npm run dev`

## 📊 Доступные страницы дашборда

После открытия дашборда вы сможете перейти на:

- **/** - Главная страница со статистикой
- **/users** - Управление пользователями
- **/transactions** - Транзакции
- **/positions** - Позиции
- **/tasks** - Задачи

## 🔗 API Endpoints

Все API доступны по адресу: http://localhost:3001/api

- `/api/users` - Пользователи
- `/api/transactions` - Транзакции
- `/api/positions` - Позиции
- `/api/tasks` - Задачи
- `/api/referrals` - Рефералы
- `/api/settings` - Настройки
- `/api/stats/dashboard` - Статистика

## ✅ Проверка работы

Если дашборд не открывается:

1. Убедитесь, что оба сервера запущены
2. Проверьте, что порты 3001 и 5173 не заняты другими приложениями
3. Проверьте консоли на наличие ошибок
4. Убедитесь, что MongoDB доступна (185.177.73.210:27017)
